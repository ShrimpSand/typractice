# カスタム配列タイピング練習

様々なキーボード配列でタイピング練習ができるNext.jsアプリケーションです。

## 特徴

- 複数のキーボード配列をサポート
  - QWERTY
  - Dvorak
  - Colemak
  - 大西配列
- 物理キーボード配列から練習したい配列への変換
- リアルタイムのタイピング統計（WPM、正確性）
- 日本語テキストでの練習

## 使い方

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

### ビルド

```bash
npm run build
npm start
```

## 使用方法

1. 物理キーボード配列を選択（現在使っているキーボードの配列）
2. 練習したい配列を選択
3. 表示されたテキストに従ってタイピング
4. 青くハイライトされたキーが次に押すべきキーです
5. Escキーでリセット、ボタンで次の文章に進めます
